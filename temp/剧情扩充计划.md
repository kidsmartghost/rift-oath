## 🔧 JSON 格式优势

### 1. 易于编辑
```json
{
  "id": "scene_001",
  "type": "choice",
  "speaker": "",
  "dialogue": "...",
  "choices": [...]
}
```

### 2. 支持多语言
```json
{
  "dialogue": {
    "zh": "黑暗。无尽的黑暗。",
    "en": "Darkness. Endless darkness.",
    "ja": "闇。無限の闇。"
  }
}
```

### 3. 版本控制
- Git 友好
- 可以 diff 剧情变化
- 支持分支合并

### 4. 工具支持
- 可以用 YAML 替代（更易读）
- 可以开发可视化编辑器
- 可以导入导出
