## 📝 使用流程

### 第一步：生成角色立绘

1. 使用 Midjourney 或 Leonardo.ai
2. 复制下方提示词
3. 生成 4 个方向，选最佳
4. 统一尺寸（建议 512×768 或 768×1024）
5. 去除背景（https://www.remove.bg）

### 第二步：生成场景背景

1. 使用相同 AI 工具保持风格一致
2. 尺寸建议 1920×1080 或 1280×720
3. 注意色调与角色立绘协调

### 第三步：生成 BGM

1. 使用 Suno 或 AIVA
2. 输入风格描述
3. 下载 MP3 格式
4. 统一音量（标准化到 -14 LUFS）

### 第四步：生成音效

1. 使用 ElevenLabs 或 Freesound
2. 短音效（<2 秒）
3. WAV 格式最佳

### 第五步：导入项目

1. 图片放入 `public/images/`
2. 音乐放入 `public/audio/bgm/`
3. 音效放入 `public/audio/sfx/`
4. 更新代码引用
