## ✅ 许可检查清单

下载每个文件时，确认：

- [ ] **CC0** (Public Domain) - 最佳，无限制
- [ ] **CC-BY** (Attribution) - 可用，需署名
- [ ] **CC-BY-SA** - 可用，需署名 + 相同许可
- [ ] **CC-BY-NC** - 仅限非商用项目
- [ ] ❌ **All Rights Reserved** - 不可用

**署名格式**（如需要）:
```
音乐："曲目名" by 作者，来源：网站，许可：CC-BY
音效："音效名" by 作者，来源：Freesound.org，许可：CC0
```

2. 左侧菜单 → 音频库

4. 下载 MP3（无需署名）
