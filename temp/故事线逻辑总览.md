## 🔗 世界连接逻辑

### 穿越机制

```
穿越者（玩家）
    │
    ├── 死亡/重伤 → 灵魂穿越到下一个世界
    │
    ├── 记忆保留 → 前一个世界的记忆碎片
    │
    └── 属性累积 → 所有世界的属性累加
        │
        ├─ 基础属性：勇气、智慧、同情、决心
        ├─ 隐藏属性：腐化、混乱
        └─ 世界羁绊：各世界独立计算
```

### 时间线关系

```
【过去】
    │
    ├── 创造者分裂自己 → 创造多元宇宙
    │
    ├── 人类发明裂隙技术 → 试图逃离
    │
    └── 核战爆发 → 人类灭绝（世界 2）
    │
    │
【现在 - 游戏主线】
    │
    ├── 世界 1：黑暗契约 → 太阳不再升起
    │
    ├── 世界 2：机器觉醒 → 猎杀者清除"污染"
    │
    ├── 世界 3：诸神黄昏 → 神族灭绝
    │
    └── 世界 4：表面普通 → 隐藏魔法秘密
    │
    │
【未来 - 真结局路线】
    │
    └── 穿越者收集所有碎片 → 面对创造者
        │
        ├─【毁灭】→ 所有世界崩塌
        ├─【牺牲】→ 自己分裂拯救世界
        ├─【守护】→ 成为新守护者
        ├─【创世纪】→ 创造新纪元（真结局）
        └─【轮回】→ 带着记忆重新开始（真结局）
```

## 📊 属性影响逻辑

### 属性阈值与剧情分支

```
勇气（Courage）
├─ ≥30：解锁部分战斗选项
├─ ≥60：解锁【领袖】选项
└─ ≥70：NPC 态度改变（称赞坚定）

智慧（Wisdom）
├─ ≥40：解锁调查/说服选项
├─ ≥50：看穿部分真相
├─ ≥60：解锁特殊对话变体
└─ ≥70：获得额外信息

同情（Compassion）
├─ ≥50：解锁和平解决选项
├─ ≥70：解锁净化能力
└─ ≥70 + 智慧≥60：真结局条件

决心（Determination）
├─ ≥30：坚持自己的选择
├─ ≥60：解锁守护者路线
└─ ≥70：真结局条件

腐化（Corruption）【隐藏】
├─ ≥20：解锁黑暗选项
├─ ≥30：NPC 警惕
├─ ≥50：堕落结局路线
└─ ≥70：毁灭者结局

混乱（Chaos）【隐藏】
├─ ≥20：看到幻象
├─ ≥40：解锁轮回结局
└─ ≥50：现实认知模糊
```

### 羁绊系统

```
世界羁绊（每世界独立）
├─ ≥30：NPC 友好态度
├─ ≥40：真结局条件
├─ ≥50：解锁特殊对话
└─ ≥100：完美结局条件
```

## 📝 剧情主题

### 核心主题

1. **存在意义** - 机器拥有人类灵魂，算什么？
2. **自由意志** - 命运是注定还是可以选择？
3. **牺牲与拯救** - 为了多数人可以牺牲少数吗？
4. **记忆与身份** - 失去记忆还是同一个人吗？
5. **创造与毁灭** - 神创造世界是对是错？

### 各章主题

- **第一章**：觉醒与接受（我是谁？）
- **第二章**：归属与羁绊（我在哪属于？）
- **第三章**：责任与力量（半神的 burden）
- **第四章**：日常与异常（平凡中的非凡）
- **第五章**：真相与选择（面对创造者）
