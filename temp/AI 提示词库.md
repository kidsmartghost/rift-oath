## 📋 使用指南

### 1. 角色立绘生成

```bash
# Midjourney 使用示例
/imagine prompt: anime style female knight, holy paladin... --ar 2:3 --v 6

# 生成后：
1. 选择最佳版本（U1-U4）
2. 放大（Upscale）
3. 去除背景（remove.bg）
4. 统一尺寸（512×768）
5. 保存为 PNG
```

### 2. 场景背景生成

