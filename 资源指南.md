## 🖼️ 添加背景图片

### 方法 1：使用免费素材网站

推荐网站：
1. **Unsplash** - https://unsplash.com/ （高质量免费照片）
2. **Pexels** - https://www.pexels.com/ （免费股票照片）
3. **Pixabay** - https://pixabay.com/ （免费图片和插画）

搜索关键词：
- **永夜王国**：`gothic church`, `dark castle`, `moonlight night`
- **机械废土**：`cyberpunk city`, `neon ruins`, `futuristic废墟`
- **神弃之地**：`ancient ruins`, `floating mountains`, `mythology temple`
- **学园都市**：`modern school`, `japanese classroom`, `library`
- **初始之境**：`white void`, `abstract light`, `minimalist`

### 方法 2：使用 AI 生成

推荐工具：
- **Stable Diffusion**（本地部署）
- **Midjourney**（Discord）
- **DALL-E 3**（Bing Image Creator）

提示词示例：
```
Gothic church interior, broken stained glass, moonlight streaming through windows, dark atmosphere, cinematic lighting, 1920x1080
```

### 使用方法

1. 下载图片到 `src/assets/images/backgrounds/`
2. 在 `script.js` 中为每个世界添加 `backgroundImage` 字段：
```javascript
{
  id: 'world1',
  name: '永夜王国',
  backgroundImage: '/src/assets/images/backgrounds/world1_church.jpg',
  // ...
}
```

3. 在 `Game.vue` 中添加背景图显示：
```vue
<div class="background-image" :style="{ backgroundImage: `url(${currentWorldBgImage})` }"></div>
```

## 🎵 添加 BGM 音乐

### 免费音乐网站

1. **YouTube Audio Library** - https://www.youtube.com/audiolibrary/
2. **FreeSound** - https://freesound.org/
3. **Incompetech** - https://incompetech.com/ （Kevin MacLeod）
4. **Bensound** - https://www.bensound.com/

### 推荐风格

| 世界 | 风格 | 乐器 |
|------|------|------|
| 永夜王国 | 黑暗史诗 | 管风琴、弦乐、合唱 |
| 机械废土 | 赛博朋克 | 合成器、电子节拍 |
| 神弃之地 | 神话史诗 | 竖琴、长笛、管弦乐 |
| 学园都市 | 日常轻松 | 钢琴、吉他 |
| 初始之境 | 空灵神秘 | 环境音、合唱 |

### 使用方法

1. 下载音乐到 `src/assets/audio/bgm/`
2. 使用 `<audio>` 标签或 Web Audio API 播放

## 🎭 添加角色立绘

### 免费角色素材

1. **itch.io** - https://itch.io/game-assets （搜索 `character portrait`）
2. **OpenGameArt** - https://opengameart.org/
3. **DeviantArt** - https://www.deviantart.com/ （注意版权）

### AI 生成角色

提示词示例：
```
Anime style female knight, silver armor, long blonde hair, serious expression, white background, character portrait, 400x600
```

### 使用方法

1. 保存为 PNG（透明背景）到 `src/assets/images/sprites/`
2. 在场景中添加 `characterSprite` 字段
3. 在 `Game.vue` 中显示立绘
